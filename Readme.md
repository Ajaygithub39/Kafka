###### Clone The Repository

    `git clone https://github.com/skshukla/KubernetesSample.git`

    `cd KubernetesSample`

    `mkdir -p /tmp/postgres`

    `chmod +x ./scripts/*.sh`



###### Run Postgres

    `./scripts/run_pg.sh`


###### Run Deployment/Services and Ingress

    `/scripts/run.sh`
